## ⚠️ 常见问题

### 1. 推送失败？

**检查清单**：
- [ ] 网络连接正常
- [ ] PAT Token有效（包含repo权限）
- [ ] 远程仓库地址正确
- [ ] 有推送权限

**解决方法**：
```bash
# 重新配置PAT
git config --global credential.helper osxkeychain

# 删除旧的凭证，重新输入
git push  # 会要求重新输入
```

### 2. 数据不更新？

**原因**：GitHub Pages缓存

**解决方法**：
- 等待2-3分钟
- 清除浏览器缓存（Cmd+Shift+R）
- 使用无痕模式访问

### 3. Excel转JSON失败？

**检查**：
- Excel文件格式正确
- 表头字段完整
- 已安装 openpyxl：`pip3 install openpyxl`

### 4. 权限问题？

**如果没有 18310258538-max 账号的权限**：

选项1：请原维护者添加你为协作者
```
Settings → Collaborators → Add people
```

选项2：Fork仓库到你自己的账号
```bash
# 推送到你的仓库
git remote set-url origin https://github.com/zhangzhiyu10/yanzhisuren.git
```
